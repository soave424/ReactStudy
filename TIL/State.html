State

State=리액트 컴포넌트의 상태-변경 간으한 데이터:개발자가 직접 정의하여 사용, 렌더링이나 데이터 흐름에 사용되는 값만 state 포함시켜야만 한다. 변경될 때 불필요한 경우 성능저하될 수 있기 때문이다. (=>JS 객체) 직접 직접 수정할 수는 없다.(가능하지만 하면 안 된다. setState 함수 이용) constructor함수로 생성함. 생성자 코드 this.state에서 정의함. (->class Component) useState라는 Hook에서 정의(->Function Component)

리액트 LifeCycle : 계속 존재하는 것이 아니라 시간의 흐름에 따라 계속 생성되고 업데이트 되다가 사라진다.
